# Sakila SQL Practice

## Project Overview
This project contains SQL queries written using the Sakila sample database.

## Objective
Analyze the database and answer analytical questions using:
- JOIN
- GROUP BY
- COUNT
- ORDER BY
- LIMIT

## Query 1: Top 5 Actors by Number of Films

This query identifies the actors who have appeared in the highest number of films.

### Skills Demonstrated
- Table joins
- Aggregation
- Grouping
- Sorting
- Data analysis thinking

## Database Used
Sakila Sample Database (MySQL)
